Con el paso del tiempo, se vio la necesidad de mejorar el aspecto de las páginas web, esto con la intención de volverlas mucho más intuitivas, así como atractivas para el ojo humano. Para poder solucionar ello, se creó la extensión o complemento Cascading Style Sheets, mejor conocido como CSS. Este recurso fue creado en 1996 por la compañía W3C, en Noruega. CSS se escribe dentro de HTML, y nos ayuda a crear sitios web más completos visualmente, así como proporcionar un ambiente más organizado.
